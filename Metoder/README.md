# Metoder

Portabla arbetsmetoder för det här projektet (och återanvändbara i andra).

| Fil | Vad |
|---|---|
| `METOD-CLI-PARALLELL.md` | Köra Claude Code CLI som parallell bug-jägare/andra-ögon via MCP-bryggan |
| `METOD-ANVANDARTEST-UI.md` | AI-personas som fritt navigerar appen i simulatorn via idb och hittar UX-fel |

Varje metod är även kodifierad som en skill i `~/.claude/skills/`
(`cli-parallell-arbetare` resp. `ux-personas-test`).
